Overview

User Can Access His Account Only From Specified MAC Address

Features

Admin Access

Admin can set MAC Address to each users.

User Restriction

Users can access their account from allowed MAC address.

Restricted Login

Accessing system from a non-specified MAC Address will restrict the user login.

Warning

A warning message will be displayed.

MAC Address

If there is no Mac is set to user acount that means there is not any restriction by MAC address. MAC address can be set on users form view for each users
